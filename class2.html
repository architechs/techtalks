A website is a way to present your content to the world, using HTML and CSS to present that content & make it look good.

CSS: What is it?

CSS = Cascading Style Sheets

CSS is a "style sheet language" that lets you style the elements on your page.

CSS works in conjunction with HTML, but is not HTML itself.

CSS: What can it do?

All colored text, position, and size

CSS: What does it look like?

The CSS Rule

The CSS Rule


selector {
  property: value;
}

A block of CSS code is a rule.

The rule starts with a selector.

It has sets of properties and values.

A property-value pair is a declaration.

CSS Syntax

Declarations: Property and value of style you plan use on HTML element.

Declarations end with a semicolon

Declaration groups are surrounded by curly brackets.


selector {
  property: value;
  property: value;
  property: value;
}

Selector: Element


p {
  property: value;
}

Selects all paragraph elements.


img {
  property: value;
}

Selects all image elements.

Selector: ID


#footer {
  property: value;
}

Selects all elements with an id of "footer".


<p id="footer">Copyright 2011</p>

The associated HTML.

Selector: Class


.warning {
  color: red;
}

Selects all elements with a class of "warning".


<p class="warning">Run away!</p>

The associated HTML.

IDs vs. Classes

ID -- Should only apply to one element on a webpage. I.E. A webpage only has one footer.

The "#" is how you tell CSS "this is an id."

Class -- Many elements can have the same class. I.E. There can be many warnings on one webpage.

The "." is how you tell CSS "this is a class name."

Selector: Descendant


p em {
  color: yellow;
}

Selects all em elements that are within a paragraph


<p>This is <em>important.</em></p>

The associated HTML.

Property Values

Each property can have one or more comma separated values.


p{
  color: white;
  background-color: red;
  font-family: Arial, sans-serif;
}

Property: Color

The color property changes the color of the text.


p {
  color: red;
  color: #ff0000;
  color: rgb(255, 0, 0);
}

Color name

Hexadecimal value

RGB value

The 17 standard colors are: aqua, black, blue, fuchsia, gray, grey, green, lime, maroon, navy, olive, purple, red, silver, teal, white, and yellow.

Pro Tip: Color

Check out W3 Schools Color Picker to select any color you want.

When you've found a color you like copy the hexadecimal value below the color block. Don't forget the '#' in front of the value!

Property: Background-color

The background-color property changes the color of the background.


p {
  background-color: black;
  background-color: #000000;
  background-color: rgb(0,0,0);
}

Property: Font-family

The font-family property deﬁnes which font is used.


p {
  font-family: "Times New Roman";
  font-family: serif;
  font-family: "Arial", sans-serif;
}

Specific font name

Generic name

Comma-separated list

Property: Font-size

The font-size property speciﬁes the size of the font.


p {
  font-size: 12px;
  font-size: 1.5em;
  font-size: 100%;
}

Pixels

"em"

Percentage

Property: Fonts (shorthand)


p {
  font-style: italic;
  font-weight: bold;
  font-size: 10px;
  font-family: sans-serif;
}
OR 

p {
  font: italic bold 10px sans-serif;
}

Connecting CSS to HTML

3 ways

"Inline"

"Embedded"

"External"

Connecting CSS to HTML: Inline


  <p style="color:red">Some text.</p>

Uses the HTML attribute style.

Difficult to use in large projects

Not preferred.

Connecting CSS to HTML: Embedded


<head>
  <style type="text/css">
    p {
      color: blue;
      font-size: 12px;
    }
  </style>
</head>

Inside <head> element.

Uses <style> tag.

Can only be used in one html file

Connecting CSS to HTML: Linked


<head>
  <link rel="stylesheet" type="text/css" href="style.css">
</head>

Shared resource for several pages.

Reduced ﬁle size & bandwidth

Easy to maintain in larger projects.

Preferred by developers everywhere!

Let's develop it

  • Create a new .css file
  • Add a link to the file in the head of the HTML page made last time
  • Add styles to change the colors, background colors or fonts of different parts of the content
  • Try using ids and classes to change specific elements

Cascading

Styles "cascade" down until changed


p{
  color:blue;
  font-family: 'Helvetica';
}
.red{
  color:red;
}
#special{
  font-family: Arial;
}


  <p>Paragraph</p>
  <p class ="green">Paragraph</p>
  <p class ="red">Paragraph</p>
  <p class = "red" id ="special">Paragraph</p>

Exercise:
Building Your Personal Website

File Structure

Let's keep our files organized and easy to work with. Create an index.html file and a style.css file in the same directory. For simplicity, we'll also put any images in this directory as well.

HTML

Let's start with the HTML file. We'll need to add all of the necessary tags to start off.


          <!DOCTYPE html>
          <html>
          <head>
            <title>Michael Scott</title>
            <link rel="stylesheet" type="text/css" href="style.css">
          </head>
          <body>
          
          </body>
          </html>

Let's also add the title of the page (can be whatever you want) and link our css file. Remember, this is the preferred way to include css in your webpages!

Adding a Header

Now let's start adding some content to the page.

  1. Start by making a div and giving it the class "header."
  2. Add a picture of yourself (or anything you like, just find one online), and make a div with the class "header-info"
  3. Let's also put in an h1 tag with the ID "page-title" since there is only one page title (This can be your name or any other title you want the page to have).
  4. Finally, put an h2 tag with a description (student, dancer, world's best boss, etc.) and then a paragraph tag with a little blurb about yourself. This and the h2 tag should all be in the "header-info" div. 

          <body>
            <div class="header">
              <img src="MichaelScott.jpg">
              <div class="header-info">
                <h1 id="page-title">Michael Scott</h1>
                <h2>World's Best Boss</h2>
                <p>My name is Michael Scott and I love Dunder
                   Mifflin Paper Company. I also like waking
                   up to the smell of bacon.</p>
              </div>
            </div>
           </body>

Your page should look something like this.

About Me

  1. Make a div with the class "about-me"
  2. Now let's add another picture of ourselves or whatever you like, and an h2 tag that says "About Me" (or whatever you want).
  3. Add a paragraph with some information so visitors of your page can get to know you.
  4. And the last thing for this div: Add an unordered list (or ordered) with some of your interests, hobbies, favorite food, etc.

            <div class="about-me">
              <img src="MichaelScott2.png">
              <h2>About Me</h2>
              <p>I am the Regional Manager of the Scranton branch of 
              paper distribution company Dunder Mifflin Inc.
              I am of English, Irish, German, and Scottish
              ancestry. I'm also two-fifteenths Native American.</p>

              <ul>This is my space. Here are some of my interests
                <li>Writing Song Parodies</li>
                <li>Playing Ice Hockey</li>
                <li>Wearing Jeans</li>
                <li>Chrysler Cars</li>
              </ul>
            </div>

Your page should look something like this.

Adding a Quote

Now for the last part of the html for our page. Let's add a div with the class "quote" (this will go outside the "about-me" div. Then put in a paragraph with your favorite quote (or just say something witty).


            <div class="quote">
              <p>"Would I rather be feared or loved? 
              Easy, both. I want people to be afraid of
              how much they love me." <em>-Me</em></p>
            </div>

CSS

Now we have a webpage but it's plain and boring. This is where CSS comes in!

style.css

Some preliminary stuff: The CSS below is to ensure everything will display properly and also sets the main font to Verdana. Open up style.css and type the code below


            * {
                padding: 0;
                margin: 0;
            }

            html, body {
                height: 100%;
                font-family: Verdana, sans-serif;
            }

Let's Style That Header Div

  • I want the background to be a different color and the text to be white
  • I also want it to cover 80% of the page and have a border on the bottom
    • Play around with some of these values and colors. Maybe you want a full border around the header that is 20px thick 
    
            .header {
                background-color: #3D4045;
                color: white;
                height: 80%;
                padding-top: 5%;
                border-bottom: 6px solid #71969F; 
            }
    Remember we use the "." selector to select classes

Floating

Let's have the image and text be side by side.
To do this we use float: left.
Let's also push the image more into the center with margin-right and put some padding between the image and the text. 


              .header img {
                  float: left;
                  margin-left: 25%;
                  padding-right: 4em;
              }
Here we see the descendant selector.
.header img will select all img tags inside the header div

header-info

Let's style the header-info div.
Remember this is the div with our header tags and a paragraph.
We'll push it in more to the center with margin-right, add some space to push it away from the image a little bit, and set our font size.


              .header-info {
                  margin-right: 25%;
                  margin-top: 8%;
                  padding-left: 4em;
                  font-size: 1.2em;
              }

Selecting an ID

We can select that h1 tag that we gave an ID of "page-title" with the "#" selector. Let's change the color of the header.


            #page-title {
                color: #959799;
            }
Don't be afraid to choose your own colors. Remember this is your webpage!

Your page might look something like this

Don't be discouraged if your page doesn't quite look like this. Everyone will have different sized images so just play around with some of the percentages until you get a layout you like

If you're having trouble with your image you can manually set its width and height with css. A good starting point is height: 600px and width 450px. 


            img {
                width:  450px;
                height: 600px;
            }
Adjust these values until you find a good width and height for your image!

About-Me Div

The styling for this will be very similiar to the header div.
We'll display everything as inline-block that way this entire div will be on its line.
We'll also set the font size and push it down a little from the header div with margin-top. 


              .about-me {
                  display: inline-block;
                  margin-top: 5%;
                  font-size: 1.2em;
              }

Layout for About Me

We'll be using float left again to get our header, paragraph, and list right next to our image. Let's also add some padding again to give everything some breathing room, and margins to push in our content as we did before.

             .about-me img {
                float: left;
                height: 600px;
                width: 450px;
                padding-right: 7.5%;
                margin-left: 20%;
              }
              .about-me p {
                padding-left: 7.5%;
                padding-bottom: 2%;
              }
              .about-me h2 {
                font-size: 2em;
                padding-top: 1%;
                padding-bottom: 2%;
              }

Styling Lists

We can change the look and positioning of lists as well.
First, to change the bullet points to squares, we'll use list-style-type: square.
Let's also push in the bullet points with list-style-position: inside.
Each list item will get some padding so they aren't too bunched up.


              ul {
                  font-size: 1.2em;
                  color: #6a727b;
                  list-style-type: square;
                  list-style-position: inside;
              }

              li {
                  padding-top: 5px;
              }

Your page might look something like this

This is only showing the About Me section. You should still have your header above this. Again, don't be discouraged if your page doesn't quite look like this. Remember that you can set the width and height of your images manually. Change some of the percentages and colors and make this page your own!

Almost There

We've got one more section to style and this one's easy! We'll start with styling the "quote" div.

  • Push the div down from the about me div with margin-top
  • Add a background color and font color
  • Align the text to the center and set a font size
    • This will apply to all text within the "quote" div 

            .quote {
                margin-top: 5%;
                background-color: #71969F;
                color: white;
                text-align: center;
                font-size: 2em;
            }

Quote Style Cont.

Let's use the descendant selector to select the paragraph inside the "quote" div.
Let's push it in on the sides with margin-right and left and add some padding to give it a big color background.


              .quote p {
                    margin-left: 25%;
                    margin-right: 25%;
                    padding-top: 7%;
                    padding-bottom: 7%;
              }
